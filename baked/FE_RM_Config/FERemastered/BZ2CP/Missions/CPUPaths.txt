List of path points that the CPU will use to build, attack, and defend.
These are not necessary for a map to work with FE IA/MPI, but will greatly improve the CPU experience.

_6 is the team number for team specific paths (6-10 for MPI cpu teams)
if suffixed by a number, that number is the slot number (for uniqueness, eg gtow4_6 will have 4 gtow position for team 6: gtow1_6 .. gtow4_6)

Defaults: 
- stage# // team agnostic but needs a number for uniqueness, staging points for units to regroup before attacking
- hold4_6 // forward defense positions (where turrets will be deployed)
- gtow4_6 // forward base locations (where guntowers will be built)
- Recycler_6 // point where the recycler will spawn and the CPU will deploy it
- TurretEnemy3_6 // point where up to three turrets will spawn
